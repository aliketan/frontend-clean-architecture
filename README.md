<p>3. Change directory to the project</p>

```
cd frontend-clean-architecture-template
```

<p>4. Install node modules</p>

```
npm install
```

<p>5. Run the project</p>

```
npm run start
```

<p>6. Compile for development</p>

```
npm run dev
```

<p>7. Compile for production</p>

```
npm run build
```
